# Amazon_Vine_Analysis
For this week's challenge, we analyzed the amazon vine program to determinate if this program has a bias toward positive reviews.

## Reviews: 
### Vines reviews: 94
### Non-Vine reviews 40,471

## Total number of 5-star reviews
### Vines reviews: 94

# Summary 

51% of the reviews were in the Vine program. At the same time, 39% of the reviews were not in the program. This indicates a bias towards positive reviews in the Vine Program.
